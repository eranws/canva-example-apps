## Run
Port 5001 is used in the examples. Configure `PORT=5001` as env variable or set immediate before `serve` command:
```
PORT=5001 serve
```

cd into directory of your choosing and then `serve` or run local path:

``
PORT=5001 serve 02-html-only 
``
